Beautiful HTTP Client

Never use CURL again. Store, organize, and execute API requests in a beautiful and elegant interface.

    

"Insomnia has replaced all my curl scripts, and it's so simple to use. No more copy/pasting or trying to remember command line syntax."

Brad Van Vugt, co-founder, sendwithus

Flexible HTTP Request Editor

Define the URL, request body, headers, and more all in one place. Define context variables for your requests and use Swig Templating to insert and format data.

Organize and Categorize Requests

Insomnia lets you organize requests into groups. Collapse and expand groups, drag-and-drop requests, and import and export your data.

Basic Auth and Custom Headers

Add Basic Authentication or custom headers to any request. Basic Auth is the standard authentication method for APIs like Stripe, Twilio, and Elasticsearch.

Keyboard Shortcuts and Request Switcher

Almost all features of Insomnia can be done without a mouse. Use your keyboard to create and execute HTTP requests, switch requests, and much more.
